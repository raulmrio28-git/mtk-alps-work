# avd

## Requirement

1. Android SDK
2. Java JDK SE 6
3. MTK Addon
4. Build image

## How to build

Build banyan_addon, must copy the project to wd410 and fix for proper compile (hwm missing, make tool fix)

## Use

Copy result image to addon folder
